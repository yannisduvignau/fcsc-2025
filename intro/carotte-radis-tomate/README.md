Enoncé : Mangez cinq fruits et légumes par jour !

```bash
    2 fichiers (txt et py)
```

Solution :
```bash
    python3 -m venv env
    source env/bin/activate
    echo "pycryptodome\nsympy" > requirements.txt
    pip install -r requirements.txt
    python3 decode.py
    deactivate
```
